Manual de Instrucciones

Esta aplicación está diseñada para la gestión de cartones y partidas de Bingo 75. Uno de los conceptos clave en la aplicación es el SERIAL, que representa un conjunto o lote de cartones.

Instrucciones para Gestionar Seriales

  • Se mostrará una tabla con todos los seriales existentes, incluyendo columnas de Nombre, Tamaño, y Fecha de Creación.
  • Se puede filtrar por:
    • Tamaño: De más cartones a menos.
    • Fecha de Creación: De más antigua a más reciente.
  • Se puede buscar por el Nombre de los seriales.

Instrucciones para Gestionar Patrones

  • Se mostrará una lista de todos los patrones existentes con su Nombre y una previsualización del mismo.

Funcionalidades

  • Añadir nuevos patrones mediante un modal que incluye un tablero 5x5 para definir el diseño.
  • Editar el Nombre y el contenido del patrón existente mediante un modal interactivo.
  • Eliminar patrones existentes.

Instrucciones para Gestionar Historial de Partidas

  • Se mostrará una lista de partidas disponibles.
  • Al seleccionar una partida, se desplegarán los datos de cada ronda, incluyendo:
    • Ronda
    • Figura
    • Ganador
    • Monto
    • Previsualización de la carta ganadora
  • Al lado del número de la partida, se mostrará el monto total premiado en todas las rondas.

Instrucciones Crear Nueva Partida

La creación de una partida ahora se divide en 7 partes, con las siguientes secciones:

1. Configuración Inicial

  • Numero de Rondas: Configura el número de rondas que se jugarán.
  • Incluir Ronda Bonus: Activa esta opción si deseas incluir rondas bonus.
  • Tómbola Automática: Activa esta opción para usar la tómbola automática para intervalos de extracción de números.
  • Premios Compartidos: Activa esta opción si quieres que varios jugadores puedan ganar Bingo con el mismo patrón.
  • Botón Confirmar Configuración: Es obligatorio presionar este botón para guardar la configuración.

2. Adjuntar Serial

  • Se mostrará una tabla con los seriales existentes, incluyendo Nombre, Número de Cartas, y Fecha de Creación.
  • Si no hay seriales creados, puedes crear uno nuevo ingresando Nombre y Número de Cartas.
  • Selecciona un serial de la tabla y presiona el botón Adjuntar para asociarlo a la partida.

3. Adjuntar Patrones

Patrones Existentes

  • Se mostrarán los patrones con opciones de editar, eliminar y añadir a una o a todas las rondas.
  • También puedes crear nuevos patrones desde esta sección.

Patrones Seleccionados

  • Se mostrarán dropdowns para cada ronda y ronda bonus.
  • Asegúrate de agregar un patrón a cada ronda y presionar el botón Cargar para adjuntar el patrón a la ronda seleccionada.

4. Personalización y Exportación

  • Personaliza los colores de las cartas con un selector de colores o usando diseños predefinidos.
  • Verás una previsualización de la carta.
  • Exporta las cartas en lotes de 150 en formato PDF. Si hay más de 150, se dividirán automáticamente en varios archivos.
  • Haz clic en el botón "Cambiar Exportación" para ver y previsualizar las cartas.

5. Venta de Seriales

  • Se muestra una tabla con todos los cartones disponibles, con su estado (Rojo - No vendido, Verde - Vendido, Azul - Seleccionado), número de cartón y nombre del comprador.
  • Vende cartones de uno en uno o en grupo.
  • Exporta cartas individuales o en 2x1 en formato PDF o PNG.
  • Exporta la tabla de venta en formato .CSV.

6. Adjuntar Premios

  • Define los premios para cada patrón de cada ronda, incluyendo rondas bonus.
  • ¡Recuerda presionar el botón Guardar para que los premios se guarden!

7. Generar Invitación

  • Agrega la fecha, hora, dinero en premios y la opción de oferta 2x1.
  • La invitación incluye información sobre las rondas, premios, patrones, cantidad de cartones, y más.
  • La invitación se puede personalizar con diseños predefinidos o colores manuales, y se exporta como PNG.

Botones Importantes

  • Continuar Última Partida: Si sales de la partida, puedes continuar donde la dejaste.
  • Cerrar Partida en Progreso: Para cerrar una partida y poder iniciar una nueva.
  • Deshacer Ajustes: Si estás creando una nueva partida, puedes deshacer todos los ajustes en cualquier momento.
