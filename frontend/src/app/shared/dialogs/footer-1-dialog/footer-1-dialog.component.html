Nos hemos unido con el propósito de adquirir experiencia real en el manejo de proyectos a nivel profesional, abordando desafíos técnicos y colaborativos que simulan un entorno laboral real.

  • Compromiso: Siempre damos el 100% en todo lo que hacemos, priorizando la calidad y los resultados.
  • Trabajo en Equipo: Valoramos la comunicación efectiva y la colaboración, lo que nos permite complementarnos mutuamente.

Áreas Técnicas

Nuestro trabajo abarca múltiples áreas técnicas, incluyendo:

  • Desarrollo Backend: Construcción de APIs robustas y servicios con .NET C#.
  • Desarrollo Frontend: Creación de interfaces modernas y responsivas utilizando Angular.
  • Infraestructura: Uso de Docker para contenerización de aplicaciones y despliegue eficiente.
  • Bases de Datos: Gestión de datos con MySQL, garantizando seguridad y rendimiento.
  • Pruebas de Calidad: Implementación de pruebas automatizadas y manuales para asegurar la excelencia del producto.

Tecnologías Usadas en Este Proyecto

Este proyecto ha sido desarrollado con un enfoque moderno, utilizando un conjunto de herramientas y tecnologías avanzadas:

  • .NET C#: Para el desarrollo de la lógica de negocio y la conexión con la base de datos.
  • Angular: Framework utilizado para la construcción de una interfaz de usuario interactiva y dinámica.
  • Docker: Para la creación de entornos consistentes y portables, facilitando el desarrollo y despliegue.
  • MySQL: Base de datos relacional utilizada para gestionar eficientemente la información del proyecto.
